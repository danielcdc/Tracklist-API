# Trianafy

Proyecto de AD y PSP sobre una API REST.

1. Librerías añadidas: 
    1) Nodemon
    2) Babel
    3) Dotenv
    4) Express
    5) Curl
    6) Body-parser
    7) Morgan (morgan y morgan-body)
    8) mongoose

2. Añadida estructura inicial del proyecto (18/01/2021):
    - controllers
    - middlewares
    - models
    - routes
    - services
    - .env
    - index.js
